# Documentos

Coloca aquí los documentos de tu empresa que deseas indexar.

## Formatos soportados:
- PDF (`.pdf`)
- Word (`.docx`)
- Texto plano (`.txt`)
- Markdown (`.md`)

## Ejemplo:
```
data/documents/
├── productos.pdf
├── politicas.docx
├── precios.txt
└── faq.md
```

Después de añadir documentos, reindexar desde el panel de administración.
